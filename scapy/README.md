# Scapy Lab

## Purpose
Packet crafting, sniffing, and traffic analysis experiments.

## Structure
- arp/ - ARP related scripts
- icmp/ - ICMP ping/flood experiments
- bgp/ - BGP message crafting
- vxlan/ - VXLAN experiments

## Notes
- Python 3.10+ recommended
- Run scripts with root privileges if sending raw packets
